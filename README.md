An Android sample app backed by the PokeAPI GraphQL endpoint, highlighting end-to-end Compose data flow and modern Android architecture practices.

## Highlights

- Apollo GraphQL: `.graphql` queries with codegen for type-safe data access.
- Jetpack Compose: declarative UI with navigation and state-driven screens.
- Hilt DI: centralized wiring for networking and repository layers.
- Layered architecture: clean separation across data/domain/presentation.
- Coroutines: async requests and state updates with Kotlin Coroutines.
- DataStore: persistent preferences and lightweight configuration storage.

## Tech Stack

- Kotlin + Coroutines
- Jetpack Compose + Navigation
- Apollo GraphQL (OkHttp)
- Hilt
- DataStore
